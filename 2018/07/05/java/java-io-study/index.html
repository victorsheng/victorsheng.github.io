概述

IO(Input And Output)在编程中是一个很常见的需求，IO即意味着我们的java程序需要和”外部”进行通信，这个”外部”可以是很多介质

  1. 本地磁盘文件、远程磁盘文件
  2. 数据库连接
  3. TCP、UDP、HTTP网络通信
  4. 进程间通信
  5. 硬件设备(键盘、串口等)
