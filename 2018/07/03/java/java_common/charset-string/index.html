特殊的字符集（ISO-8859-1）

ISO-8859-1是单字节字符集，是ASCII字符集的补充。通常情况下使用ISO-8859-1字符集进行字符串对象与字节化字符数据的互换操作与前述完全一致。

利用ISO-8859-1字符集，我们可以将任何一个字节数组无损保存到字符串对象中。
